    Release Notes

    Version 1.4.4 May 2026New

    • Intelligent Recorder: Capture clicks, typing, selects and scrolls in your app, then generate AI-powered, paste-ready tests with smart waits and assertions — Selenium (Java / Python / JavaScript), Playwright (JS / TS / Python), Cypress, or WebdriverIO.
    • Copy as Framework: Every locator can now be copied as wrapped framework code, not just a raw selector. New Copy as dropdown next to the Engine selector.
    • Anchor-first v2 Engine: New default locator engine extracts identity anchors (data-testid, role + accessible name, ARIA labels, stable classes) and beam-searches for the most resilient locator. Switch back to v1 from the Engine dropdown anytime.
    • Unified banner stack: All eligibility-driven announcements (update available, new features, AI credits) now live in a single section at the top of the panel — no more banners scattered above and below the controls.
    • Bug fixes: Removed duplicate XPath rows for anchor (<a>) elements where XPath by Text and XPath by Link Text used to render identically. Improved escalation from non-semantic SVG / decorative elements to their interactive ancestors.

    Version 1.4.2 December 2025

    • Multi-Provider Support: Added support for all Gemini models and OpenRouter integration.
    • Improved AI Accuracy: Enhanced optimization logic for precise, robust locators.
    • Smart Validation: AI locators are now seamlessly integrated with the validation system.
    • Enhanced UI: Modern, cleaner interface with improved accessibility.
